# Markdown Editor & Content Tools

This page provides a powerful markdown editor for creating content with real-time preview, along with tools for formatting and Hugo shortcodes.

## Features

- **Real-time Markdown Preview**: See your content as you type
- **Toolbar Buttons**: Quick formatting for headers, bold, italic, links, and more
- **Hugo Shortcodes**: Built-in support for video shortcodes
- **Responsive Design**: Works on desktop and mobile devices
